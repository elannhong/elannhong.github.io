Eisenhower Matrix

Website Link

A simple tool that helps you prioritize tasks based on urgency and importance by sorting them in a matrix. Based on the quadrant you put the task in, you can determine how to handle it:

  • Urgent and Important: Do these tasks immediately
  • Urgent and Not Important: Delegate these tasks to someone else
  • Not Urgent and Important: Schedule these tasks for a later time
  • Not Urgent and Not Important: Delete these tasks

Inspiration

I like using todo lists to organize my daily tasks, but I often struggled to decide which tasks to do first. I made this tool to help myself and others easily arrange tasks in a visual way.

The two main features of this project are creating tasks and dragging tasks around. Here's a quick explination of how each feature was implemented.

Creating Tasks

  1. A taskbox is made after the user hits the add task button, enters a task, and hits the enter button. After this is done the inputbox disappears.

    document.querySelector("html").addEventListener("keydown", e => {
    if (e.key === 'Enter' && input.value!='' && input.style.display === "block") {
        input.style.display = "none";
        taskbox.style.display = "flex";
    //...
    }}}

  2. To create a new task, a template task is cloned. The new task gets the 'taskbox' id and the task text from input.value.

    const tasktemplate = document.querySelector('.taskbox');
const newtask = tasktemplate.cloneNode(true);
newtask.classList.add('taskbox'); // ensure class is present
newtask.textContent = input.value; // set text
  3. Finally, add a drag event listener to the newtask so it can be dragged (explained below) and insert it. 
    newtask.addEventListener("mousedown", dragStart);
newtask.addEventListener("touchstart", dragStartTouch);
tasktemplate.parentNode.insertBefore(newtask, tasktemplate.nextSibling);

Dragging Tasks

  1. Start Drag: When you press and hold (mousedown) on a taskbox, the dragStart function is triggered, setting up the drag state and calculating the mouse offset. 
    2. function dragStart(e) {
    isDragging = true;
    dragTarget = e.target;
    offsetX = e.clientX - dragTarget.offsetLeft;
    offsetY = e.clientY - dragTarget.offsetTop;
    dragTarget.style.cursor = "grabbing";
    e.preventDefault();
}
  2. Move Task: As you move the mouse, the taskbox follows the cursor, updating its position based on the mouse coordinates minus the offset.
    This is done because we want the mouse to stay in the same position relative to the taskbox while the box is moved. The taskbox dragTarget postion can only be set with the coordinates of the top left corner. So to have the taskbox stay the same distance from the mouse for the entire dragging duration, we set the box's top left corner position to the mouse's position minus the inital offset when clicked. 
    3. document.addEventListener("mousemove", (e) => {
    if (isDragging && dragTarget) {
        dragTarget.style.left = (e.clientX - offsetX) + "px";
        dragTarget.style.top = (e.clientY - offsetY) + "px";
    }
    // ... check for overlap with delete box ...
});
  3. Drop Task: When you release the mouse (mouseup), dragging stops. If the task is over the delete box, it is removed. 
    4. document.addEventListener("mouseup", () => {
    if (dragTarget) dragTarget.style.cursor = "grab";
    if(isOverBlock){
        dragTarget.remove();
    }
    isDragging = false;
    dragTarget = null;
    isOverBlock = false;
    delBlock.classList.remove("hover");
});
  4. Touch Support: The same logic applies for touch events, using touchstart, touchmove, and touchend listeners for mobile devices.
